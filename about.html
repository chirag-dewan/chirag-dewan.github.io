Chirag Dewan

Security engineer with 3+ years in offensive security research, cloud security, and detection engineering. Background in zero-day vulnerability discovery across ICS environments, ML-powered fraud detection, and cloud-native security architecture. Currently SWE II at GM Financial in Dallas.

Previously an offensive security researcher at RTX/RTX BBN, where I discovered 8 zero-day vulnerabilities in ICS protocols and built AI-driven intrusion detection for embedded firmware. Before that, interned at RTX (senior cybersecurity intern) and Reata Pharmaceuticals (incident response during an active breach).

I build systems that detect abuse through behavioral patterns — not content inspection. My current research: PARALLAX, a metadata-only threat detection pipeline validated against real LANL data (0.68 AUC), and KESTREL, its infrastructure-layer companion for cloud workload anomaly detection (0.766 recall on real CloudTrail data).

Interested in the space where offensive security meets machine learning — building defenses informed by how attacks actually work. OSCP candidate. ASU CS, Summa Cum Laude.

Based in Texas. Targeting Boston by late 2026.
