MCP-Poison-Bench
held-out Δ ≈ 0 · rug_pull 1.00 → 1.00 · worst model: gpt-4o-mini

The client-side tool_transform defense's apparent win was circular: on a held-out payload set authored to trip zero rules the reduction vanishes, and rug_pull (result-borne injection) bypasses the defense by design. PARALLAX
0.45 → 0.68 AUC · 16.9M LANL auth events · 87/500 entities compromised
Metadata-only behavioral detection of model-extraction abuse, with no prompt or payload inspection. A 14-feature bank scored worse than chance on real authentication data because 7 features had memorized a synthetic attacker's silhouette; collapsing it to a single change-from-baseline signal recovered the detector. Its three documented evasions ship with it.
